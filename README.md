# 모아트립 MoaTrip

> 여행비를 함께 기록하고, 나누고, 정산하는 모바일 공동 여행 장부

**모아트립**은 여러 명이 함께 떠난 여행에서 각자 결제한 금액을 기록하고, 최종적으로 누가 누구에게 얼마를 보내야 하는지 간편하게 확인할 수 있는 모바일 웹앱입니다.

🔗 **바로가기:** https://moatrip.vercel.app

---

## 주요 기능

### 여러 여행 관리

- 여행별 이름, 지역, 기간 설정
- 여러 여행 생성 및 전환
- 여행 보관 및 복원
- 여행별 참여자와 지출 내역 분리
- 여행별 통화 설정
  - KRW
  - JPY
  - USD
  - EUR

### 공동 지출 기록

- 결제자 선택
- 참여자 균등 분배
- 참여자별 직접 금액 입력
- 카테고리, 메모, 날짜 기록
- 영수증 사진 첨부
- 지출 수정 및 삭제
- 비슷한 지출 중복 등록 경고

### 정산

- 참여자별 결제액과 부담액 계산
- 받을 금액과 보낼 금액 표시
- 송금 횟수를 줄인 정산 결과 제공
- 송금 완료 상태 공동 저장
- 장부 마감 및 다시 열기

### 공유

- 여행별 편집 링크 생성
- 보기 전용 링크 생성
- 링크 복사
- 카카오톡, 문자 등 모바일 공유창 지원
- 초대 링크 접속 시 해당 여행만 표시
- 기기별 사용자 이름 기억

### 백업 및 관리

- CSV 내보내기
- JSON 전체 백업
- JSON 백업을 새 여행으로 복원
- 오래된 영수증 일괄 정리
- 삭제한 지출 영구 정리
- 카테고리별 지출 통계

### 모바일 사용성

- 모바일 중심 반응형 UI
- 홈 화면 설치가 가능한 PWA
- 오프라인 캐시
- 실시간 동기화
- 실시간 연결 실패 시 자동 동기화
- 새 버전 업데이트 안내
- 저장 중, 저장 완료, 오프라인 상태 표시

## 사용 방법

1. 여행 이름, 지역, 기간을 설정합니다.
2. 함께 여행한 참여자를 추가합니다.
3. 각 지출의 결제자와 분담 대상을 선택해 저장합니다.
4. 여행이 끝나면 정산 화면에서 송금 내역을 확인합니다.
5. 여행을 함께 기록하려면 상단의 **공유** 버튼으로 초대 링크를 보냅니다.
6. 모든 기록이 끝나면 장부를 마감하거나 여행을 보관합니다.

## 화면 구성

| 메뉴 | 설명 |
|---|---|
| 홈 | 총지출, 개인 결제액, 부담액, 받을 금액, 최근 지출 확인 |
| 내역 | 여행의 전체 지출 검색 및 확인 |
| 추가 | 새로운 지출 등록 |
| 정산 | 참여자별 정산 결과와 송금 완료 상태 확인 |
| 설정 | 여행 정보, 참여자, 장부 마감, 백업 및 관리 |

## 로컬에서 실행하기

이 프로젝트는 정적 웹앱이므로 간단한 로컬 서버로 실행할 수 있습니다.

### VS Code Live Server

1. 저장소를 내려받습니다.
2. VS Code에서 프로젝트 폴더를 엽니다.
3. `index.html`을 Live Server로 실행합니다.

### Python

```bash
python -m http.server 8000
```

그다음 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8000
```

`file://` 방식으로 직접 열면 Service Worker와 일부 브라우저 기능이 제한될 수 있습니다.

배포에 필요한 파일:

## Appwrite 설정

앱은 Appwrite Cloud를 공동 저장소로 사용합니다.

필요한 구성:

- 프로젝트
- 데이터베이스
- 여행 테이블
- 참여자 테이블
- 지출 테이블
- 영수증 Storage 버킷
- 실제 배포 주소가 등록된 Web Platform

현재 구조에서는 여행별 추가 설정을 `dataJson`에 함께 저장하므로, 새로운 기능을 사용할 때 별도의 열을 추가하지 않아도 됩니다.

## 권한 및 보안 안내

모아트립은 가족, 친구, 연인처럼 서로 신뢰하는 사람끼리 간편하게 사용하는 것을 목표로 만든 공개형 공동 장부입니다.

현재 Appwrite 데이터와 영수증은 로그인 없이 모두가 읽고 수정할 수 있도록 설정되어 있습니다.

따라서 다음 사항을 확인해 주세요.

- 초대 링크는 신뢰할 수 있는 사람에게만 공유해 주세요.
- 주민등록번호, 카드번호, 계좌 비밀번호 등 민감한 정보는 기록하지 마세요.
- 보기 전용 링크는 화면에서 수정 기능을 숨기는 편의 기능이며, 계정 기반 보안 권한은 아닙니다.
- 외부에 공개되면 곤란한 영수증은 여행 종료 후 정리하는 것을 권장합니다.
- 중요한 여행 데이터는 JSON 백업으로 별도 보관하는 것이 좋습니다.

## 데이터 동기화

- Appwrite Realtime을 이용해 다른 기기의 변경 사항을 빠르게 반영합니다.
- 실시간 연결이 끊기면 일정 시간마다 자동으로 다시 확인합니다.
- 동시에 같은 지출을 수정하면 최신 수정 여부를 확인하고 충돌 경고를 표시합니다.
- 송금 완료 상태는 최신 데이터를 기준으로 병합해 저장합니다.

## PWA 설치

모바일 브라우저에서 접속한 뒤 홈 화면에 추가하면 일반 앱처럼 사용할 수 있습니다.

### Android Chrome

```text
브라우저 메뉴 → 홈 화면에 추가
```

### iPhone Safari

```text
공유 버튼 → 홈 화면에 추가
```

새 버전이 배포되면 앱 안에서 업데이트 안내가 표시될 수 있습니다.
